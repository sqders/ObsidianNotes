# Определение
**Инверсия управления - это подход к программированию** (такой же как ООП, функциональный и т.д.), который позволяет *контейнеру или фреймворку* создать объект **по необходимости**. Теперь **не мы управляем жизненным циклом объекта, а предпочтительно - контейнер**, который создает объекты созданных нами классов по необходимости и в нужных ситуациях
